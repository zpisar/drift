# Drift Feature Roadmap

## Phase 1: Skill Readability

- Perfect dodge bonus for last-moment switches
- Near-miss feedback with very subtle visual response
- Light score feedback for skillful play
- Make perfect-dodge feedback text larger and longer lasting
- Make the score pulse more intense and easier to notice

## Phase 2: Run Shape

- Milestone markers every 10 seconds
- Slight wave rhythm in obstacle pacing
- Game-over messaging for personal bests and rank chase

## Phase 3: Retention

- Leaderboard rank delta messaging
- Minimal combo or flow multiplier
- Refined restart flow for faster repeat runs

## Design Rules

- Keep the game one-button
- Preserve the minimalist visual language
- Add feedback, not clutter
- Favor fairness and mastery over randomness
